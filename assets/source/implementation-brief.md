# Codex implementation brief

Build a responsive one-page website based on `website-mockup-desktop-mobile-final.png`.

Use:

- `copy.md` for final copy
- `visual-style-guide.md` for the design system
- `matei-chewbacca-original.jpeg` for the hero image

## Non-negotiable composition

The portrait must bridge the red hero and cream About section.

One foot should visibly sit on the cream section while the rest of the body remains on red.

Preserve the full body and both arms.

## About section

Keep it clean.

Do not add:

- crowd image
- team image
- stock photography
- decorative illustration

Use only the three achievement statements with generous spacing.

## Header

Show only:

> Matei Psatta

No navigation menu.

## Do not add

- About / Manifesto / Contact links
- gradients
- glassmorphism
- rounded SaaS card grids
- recruiter language
- “Chemistry matters”
- “People not resources”
- “No recruiter spam”
- “No job-board noise”
- “No fee”

## Behavior

Reuse the existing accessible tab and form logic where practical.

Maintain:

- visible keyboard focus
- semantic headings
- correct tab roles
- reduced-motion support
- adequate contrast
- large touch targets
